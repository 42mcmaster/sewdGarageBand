# Run Rudolph Run

**Artist:** Chuck Berry  
**Composers:** Credited variously Berry or Berry, Marks  
**Key:** C  
**Tuning:** Standard (E A D G B E)  
**Capo:** No capo

---

**Note:** This song reminds many of "Little Queenie." There is a slight difference between the first line of the choruses - there is a change to F in the first chorus which is missing in later ones (shown in brackets).

---
